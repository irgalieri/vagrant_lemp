# vagrant_lemp

## Description
Vagrant File with Ubuntu 15.04 LTS, PHP 5.6 and MySQL

## Licence
GPL-3.0

## Usage
* Clone this project
```
   $> git clone git@github.com:irgalieri/vagrant_lemp.git
```
* Exec vagrant up
```
   $> vagrant up
```
* Test got to http://localhost:8080/index.php

## Mysql
* Host localhost:3306
* user: root
* pwd: vagrant
